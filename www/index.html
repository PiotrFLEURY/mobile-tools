mobile-tools.dev is a collection of tools for mobile developers.
  • Flutter project
  • Dartproject

  • camcode

    camcode is a Flutter Web plugin to scan barcodes and QR codes using javascript library
    https://pub.dev/packages/camcode

  • every_test

    every_test is a Flutter test library to run a single test multiple times tests using different parameters.
    https://pub.dev/packages/every_test

  • look

    look is a Flutter dev tool to run a single widget in hot reload mode.
    https://pub.dev/packages/look

  • nenuphar_cli

    nenuphar_cli is a command line tool to generate a Swagger for your Dart Frog server.
    https://pub.dev/packages/nenuphar_cli
